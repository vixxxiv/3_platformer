# JavaScript Платформер

Полнофункциональная игра-платформер, созданная на чистом JavaScript с использованием HTML5 Canvas.

## Функциональность

- Управление персонажем (движение влево/вправо, прыжки)
- Система физики с гравитацией
- Коллизии с платформами
- Сбор монет для набора очков
- Несколько уровней сложности
- Система жизней
- Полноценный игровой интерфейс

## Управление

- **Стрелки влево/вправо** или **A/D** - движение
- **Пробел** или **W** или **Стрелка вверх** - прыжок

## Использование нейросети

**Нейросеть**: DeepSeek
**Время выполнения**: ~4 часа

### Части игры, созданные с помощью нейросети:

1. **Базовая архитектура проекта** - структура файлов и классов
2. **Система коллизий** - алгоритмы обнаружения столкновений
3. **Игровой цикл** - реализация основного цикла игры
4. **Физика движения** - гравитация и движение персонажа
5. **Отрисовка элементов** - визуальное представление игровых объектов
6. **Обработка ввода** - система управления с клавиатуры

## Запуск игры

1. Скачайте все файлы в одну папку
2. Откройте файл `index.html` в браузере
3. Нажмите кнопку "Старт" для начала игры

## Структура проекта

- `index.html` - основной HTML файл
- `css/style.css` - стили игры
- `js/main.js` - инициализация игры
- `js/Game.js` - основной класс игры
- `js/Player.js` - класс игрока
- `js/Platform.js` - класс платформ
- `js/Coin.js` - класс монет
- `js/Collision.js` - система коллизий
- `js/InputHandler.js` - обработка ввода
- `js/Renderer.js` - отрисовка игры
